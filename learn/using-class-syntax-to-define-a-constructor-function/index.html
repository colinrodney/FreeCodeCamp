Use class Syntax to define a Constructor Function

Desmistifying the 'this' keyword

  • ES6 provides new syntax to create objects using the class syntax
  • Objects are created using the "class" keyword - NOTE: class is just syntax and not a full-fledged implementation of an object-oriented paradigm. This is different from other languages such as Java, Python, Ruby, etc...
  • Constructor functions are defined using the new keyword which creates a new instance (instantiates) an object

  • NOTE: the class keyword declares a brand new function to which a constructor is added. The constructor is invoked when the new keyword is called to create a new object.
  • UpperCamelCase shuld be used in class naming syntax in ES6
  • The constructor method: special method used to create and initialize an object created w/ a class.(Learn mode about the constructor method in the Object Oriented Programming Section.
